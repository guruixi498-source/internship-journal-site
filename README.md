# 实习与学习记录网页

这是一个用于记录实习收获、学习感悟、个人思考和文档资料的个人静态网页。

## 功能

- 新增实习、学习、思考、文档四类记录
- 文档记录支持添加外部链接
- 支持按分区筛选记录
- 支持删除记录
- 使用浏览器 localStorage 保存本地数据
- 适配桌面和手机浏览

## 本地预览

直接用浏览器打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/index.html
```
